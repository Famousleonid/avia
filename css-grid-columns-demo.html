CSS Grid - Управление шириной столбцов

1. Фиксированная ширина столбцов

Используйте конкретные значения (px, em, rem) для точного контроля ширины.
100px
200px
150px
.grid-template-columns: 100px 200px 150px;

2. Процентная ширина

Проценты относительно ширины контейнера. Общая сумма может быть меньше 100%.
25%
50%
25%
.grid-template-columns: 25% 50% 25%;

3. Fractional units (fr)

fr делит доступное пространство пропорционально. 1fr = 1 часть доступного пространства.
1fr
2fr
1fr
.grid-template-columns: 1fr 2fr 1fr;

4. Смешанные единицы измерения

Комбинируйте разные единицы: фиксированную ширину + fr для гибкости.
200px
1fr
2fr
100px
.grid-template-columns: 200px 1fr 2fr 100px;

5. minmax() - минимальная и максимальная ширина

Задайте диапазон ширины: minmax(минимальная, максимальная).
minmax(100px, 200px)
minmax(200px, 1fr)
minmax(150px, 300px)
.grid-template-columns: minmax(100px, 200px) minmax(200px, 1fr) minmax(150px, 300px);

6. auto - автоматическая ширина

auto подстраивается под содержимое элемента.
auto (содержимое)
1fr (гибкий)
auto (содержимое)
.grid-template-columns: auto 1fr auto;

7. repeat() - повторяющиеся столбцы

Создайте несколько одинаковых столбцов: repeat(количество, размер).
1fr
1fr
1fr
.grid-template-columns: repeat(3, 1fr);

8. Адаптивные столбцы

auto-fit создает столбцы автоматически, auto-fill заполняет все доступное пространство.
minmax(200px, 1fr)
minmax(200px, 1fr)
minmax(200px, 1fr)
minmax(200px, 1fr)
.grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

9. Именованные линии сетки

Дайте имена линиям сетки для более удобного позиционирования элементов.
sidebar: 200px
main: 1fr
sidebar2: 150px
.grid-template-columns: [sidebar] 200px [main] 1fr [sidebar2] 150px;

10. Grid Template Areas

Создайте макет с именованными областями для сложных компоновок.
Header (1fr 2fr 1fr)
Sidebar (1fr)
Main (2fr)
Aside (1fr)
Footer (1fr 2fr 1fr)
.grid-template-columns: 1fr 2fr 1fr; .grid-template-areas: "header header header" "sidebar main aside" "footer footer footer";

Практические советы

Когда использовать разные подходы:
  • px/em/rem: для фиксированных размеров (сайдбары, навигация)
  • %: для относительных размеров относительно контейнера
  • fr: для гибкого распределения доступного пространства
  • minmax(): для адаптивности с ограничениями
  • auto: когда ширина должна зависеть от содержимого
  • repeat(): для создания одинаковых столбцов
Полезные комбинации:
  • grid-template-columns: 250px 1fr; - фиксированный сайдбар + гибкий контент
  • grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); - адаптивная сетка карточек
  • grid-template-columns: auto 1fr auto; - логотип + контент + кнопки
  • grid-template-columns: minmax(200px, 300px) 1fr; - сайдбар с ограничениями
